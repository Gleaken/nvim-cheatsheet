# Movements

| keys | Description |
|------------|---------------|
|w | next word  |
|b | previous word  |
|u | undo  |
|crtl+r | redo  |
|v | visual mode  |
|shift + v | visual line mode  |
|_ | to begin of line (char)  |
|$ | to end of line  |
|0 | to column 0  |
|f<char> | goes forward to first occurence of a char  |
|F<char> | goes backward to first occurence of a char  |
|t<char> | goes forward to beforefirst occurence of a char  |
|T<char> | goes Backward to beforefirst occurence of a char  |
|, | repeat search backwadr  |
|; | repeat search forward  |
|I | go to beginig of line in insert mode  |
|A | go to end of line in insert mode  |
|o | go to new line and insert mode  |
|O | goto new line above cursor and insert mode  |

# Windows
Ctrl+w - to next window
Ctrl+q - close current window
